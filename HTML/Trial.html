Virtual Mouse Control

Learn More

AI Virtual Mouse. Developed an AI-based approach for controlling the mouse movement using Python and openCV with real-time camera that detects hand landmarks, tracks gesture patterns instead of a physical mouse. This Virtual Mouse Hand Recognition application uses a simple color cap on the finger without the additional requirement of the hardware for the controlling of the cursor using simple gestures and hand control. This is done using vision based hand gesture recognition with inputs from a webcam.
